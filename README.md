# annual-diagram-app
A simple web-app to display bar diagram/chart

## server
### Project setup
```
npm install
```

### Database Setup 
- Create the Database
```
npx sequelize db:create
```

- Migration
```
npx sequelize db:migrate
```

- Seed the Data
```
npx sequelize db:seed:all
```

### Run the Server
```
npx nodemon app.js
```
<br>
